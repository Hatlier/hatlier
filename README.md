# Hatlier

**ちょっとしたページを、生成AIなしでも自分で直せる。**

HTML は完成品ではなく、創作物を包む器。Hatlier（H + at + *lier*「つなぐ」）は、ブラウザだけで組み立て・書き換え・配布できる **1ファイル完結** の HTML エディタです。

## すぐ使う

- デモ: https://hatlier.github.io/hatlier/
- ローカル: [`hatlier.html`](./hatlier.html) をブラウザで開く（または [`index.html`](./index.html)）

文字をクリックして直す → ブロックを足す・並べる → 「HTML書き出し」で 1 枚の HTML を渡す。  
サーバーもインストールも不要です。

## 名前

| 要素 | 意味 |
|---|---|
| **H** | HTML |
| **at** | 「〜で」「〜に」という接続のニュアンス（Atelier の面影） |
| **lier** | フランス語 *lier* — つなぐ・結ぶ |

## 方針の要約

- 凝ったページは本職と生成 AI に任せる
- Hatlier の土俵は「ちょっとしたページを、自分の手で直して渡す」こと
- 詳細は [`SPEC.md`](./SPEC.md)
- **これからやること**は [`ROADMAP.md`](./ROADMAP.md)（画面の `vX.Y.Z` からも開けます）

## 書き出し

HTML 1枚 / ZIP / PDF / PNG / WebM / スライド動画（WebM）をメニューから選べます。
